# Glubinka MP — PublicSDK

Публичный SDK для разработчиков серверных ресурсов Glubinka MP.

## Скачать
Открой раздел **Releases** и скачай последнюю версию `GlubinkaMP-PublicSDK`.

## Что внутри
- `Lib`
- `Templates`
- `Examples`
- `Docs`

## Как начать
1. Скачай ZIP из Releases
2. Открой папку `Docs`
3. Прочитай `QuickStart.md`
4. Используй `Templates` и `Examples` как основу для своих ресурсов


## ⚖️ Правовой статус и Лицензия

Использование платформы регулируется файлами `LICENSE.md` и `TRADEMARKS.md`.

**✅ Вам разрешено:**
- Запускать собственные серверы (публичные и приватные).
- Писать, публиковать и **продавать** свои скрипты, карты и интерфейсы.
- **Монетизировать сервер** (принимать донаты, продавать VIP-статусы, подписки и внутриигровые услуги).

**❌ Вам запрещено:**
- Вскрывать, изменять или распространять закрытое ядро платформы (Core) как свой собственный продукт.
- Использовать бренд так, чтобы выдавать свой сервер или мод за **«официальный проект Glubinka MP»** без разрешения правообладателя.

*Кратко: делайте крутые серверы, зарабатывайте на них, пишите скрипты, но не ломайте ядро и не называйте себя разработчиками платформы.*
